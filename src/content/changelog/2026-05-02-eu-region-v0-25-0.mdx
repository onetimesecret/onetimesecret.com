EU region (eu.onetimesecret.com) jumped straight from v0.22.1 to v0.25.0, skipping the
gradual changes in v0.23 and v0.24 entirely. EU was the last region to migrate, completing
a staggered rollout that started with the UK in March. All regional sites can now run the
same latest version. A follow-up to v0.25.1 landed two days later with EU-specific
migration hardening.
