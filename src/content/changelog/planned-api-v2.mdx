We're working on a major update to the Onetime Secret API. The new version introduces scoped access tokens so you can create keys with only the permissions they need, built-in rate limit headers so integrations can self-throttle, and auto-generated OpenAPI documentation.

Your existing API keys will continue to work unchanged. The v2 API will be available alongside v1 during a transition period.
